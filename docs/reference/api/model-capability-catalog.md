# 模型能力分类总表

本文件只覆盖当前 `src/constants/models.ts` 中 `available: true` 的模型。

编写规则：

- 先按 `图片 / 视频 / 声音` 分类。
- 再按官方文档明确写出的能力做二级分类。
- “能做什么”与“特点”只记录官方文档、官方模型页、官方模型卡、或官方 API 文档明确写出的内容。
- 如果官方页面当前只明确到“这是某个端点 / 某个版本”，没有更多卖点说明，这里就保守记录，不补脑。
- 如某个仓库模型接的是某家族的特定端点，这里会优先写“当前仓库主端点”；家族级能力只有在官方资料明确说明时才补充。

覆盖统计：

- 图片：27
- 视频：16
- 声音：2
- 合计：45

## 维护建议

1. 这份目录只应跟随官方文档更新，不要把主观体验、社区排行、个人审美写进来。
2. 每次新增模型时，同时更新：
   - `src/constants/models.ts`
   - `docs/reference/api/model-capability-catalog.md`
   - `docs/reference/api/model-doc-monitor.snapshot.json`
3. 每周 GitHub Action 报警后，先看：
   - 官方文档是否改了模型名 / 版本 / 生命周期
   - 官方 API 列表是否出现了新的稳定 slug
   - 这份目录是否需要补新能力或删掉已退场能力
