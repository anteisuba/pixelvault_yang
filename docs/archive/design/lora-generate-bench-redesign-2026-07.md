## 1. 信息架构(方向 A 核心)

**一句话:参考物料与工作面彻底分层——台面只有一个,料架在手边。**

```
┌ 脊柱条(变薄):当前 LoRA chips+兼容点 · 底模 Select · [助手] ┐
│ [物料rail] │        中央台面(单列,~max-w-3xl)      │ [助手dock] │
│  48px      │  ① 监视器(出图预览/空态起手引导)      │  按需开    │
│  ▣ 参考    │  ② filmstrip 横条(会话结果 ≤12)      │            │
│  ▣ 词库    │  ③ 纸:触发词行/正文(补全)/负向/     │            │
│            │     警示行/配额行/忠实·比例·参考图/出图 │            │
└────────────────────────────────────────┘
```

- **中央台面 = 单一垂直视轴**:看图(①)→写词(③)→出图(③底)→结果回到(①),视线不再离轴。监视器上、纸下(暗房习惯:显影盘在眼前,记录在手边;纸贴出图钮减少手移)——【拍板项 ①,可翻转】。
- **物料 rail(左,48px 图标列)**:两个入口——「参考」「词库」。点击**覆盖式浮出**面板(~480px,压在台面左沿上方,不推挤台面;Esc/点外关闭;同刻最多开一个)【拍板项 ②:覆盖 vs 推挤,我判覆盖——台面宽度稳定是本案的魂】。
- **助手 dock(右)**:F2 现状不动(宽度记忆/拖宽),与 rail 对称成「左料右智」。
- 脊柱条保留但**变薄**:去掉独立占行的余量点(●●○○○ 移入 LoRA chips 区尾),API key 琥珀提示保留。
