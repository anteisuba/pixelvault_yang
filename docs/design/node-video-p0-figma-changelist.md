## 4. 动效规格（Motion Notes，U1 + 面板内）

全部走现有 canon：`--ease-standard cubic-bezier(0.22,1,0.36,1)`，时长 token `fast 120 / base 200 / slow 320`，reduced-motion 全局兜底。**Figma 里以注释帧交付即可，不强求原型连线。**

| #   | 对象                       | 规格                                                                                                                                                                                       |
| --- | -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| M1  | **顶栏收放**（U1）         | 收起：整条顶栏向上位移 8px + 淡出（`slow`），pill 按钮淡入（`base`，延迟 80ms）；展开反向。`AnimatePresence` 双元素交叉，不做高度塌缩                                                      |
| M2  | **助手 dock 出入场**（U1） | 桌面：自右滑入 16px + 淡入（`slow`）；移动端自底部（对齐 vaul 抽屉手感）；关闭反向缩短为 `base`。与现有宽度形变 `.node-canvas-panel-motion` 并存：出入场 = transform/opacity，尺寸 = width |
| M3  | 面板内反馈                 | 槽位插入 token 高亮褪色 300ms；OSD 段展开 = `base` 缩放 0.98→1；生成中 TC 跳动（P2 实现，注释即可）                                                                                        |
