# PixelVault — 个人 AI 画廊

用 20+ AI 模型生成图像和视频。盲投比较。永久归档。

**立即体验:** [pixelvault-seven.vercel.app](https://pixelvault-seven.vercel.app/)

![PixelVault 首页](docs/screenshots/hero.png)

---

## PixelVault 是什么？

PixelVault 是一个多模型 AI 图像与视频生成平台。选择任意模型 — GPT-Image、Gemini、FLUX、Kling、Sora 等 — 用同一个提示词生成并对比，每一件作品都会连同设置和元数据一起永久保存。

---

## 快速开始

1. **注册** — 注册即送免费积分
2. **进入 Studio** — 选一个模型，输入提示词
3. **生成** — 图像或视频即刻创建并永久保存

就这么简单。无需配置，无需 API 密钥（除非你想用高级模型）。

## 功能

### Studio — 创作图像和视频

你的创作工作台。写提示词，选模型，一键生成。

![Studio](docs/screenshots/studio.png)

- **11 个图像模型 + 10 个视频模型**，来自 6 个供应商
- **提示词增强** — 让 AI 用 5 种风格（详细、艺术、写实、动漫、LoRA）重写你的提示词
- **参考图** — 上传图片作为生成的视觉引导
- **图像逆向** — 分析任意图像，提取生成参数
- **图生视频** — 将生成的图片转化为视频片段
- **角色卡片** — 保存角色预设，确保多次生成的视觉一致性
- **视频时长** 3 秒至 120 秒，最高 1080p

### Gallery — 发现与分享

浏览其他用户的作品，获取灵感，分享你的创作。

![Gallery](docs/screenshots/gallery.png)

- 按提示词文本搜索
- 按模型、类型（图像/视频）、时间范围筛选
- 点赞和收藏喜欢的作品
- 点击图像查看完整详情 — 提示词、模型、设置
- 将画廊中的任意图像用作下一次生成的参考图

### Arena — 盲投模型对决

哪个模型才是最强的？盲投决出胜负。

![Arena](docs/screenshots/arena.png)

1. 选择 2–4 个模型，输入提示词
2. 所有模型同时从相同提示词生成
3. **在不知道哪个模型生成的情况下**投票选出最佳结果
4. 投票后揭晓模型身份和 ELO 评分变化
5. 查看**排行榜**了解模型整体排名

### Profile — 个人档案库

你的每一次生成，都在这里。

- 搜索和筛选查看所有作品
- 逐张切换公开/私密可见性
- 数据统计面板 — 总生成数、请求数等
- 支持彻底删除（含存储清理）

## 使用技巧

**自带密钥 (BYOK)** — 想用高级模型又不想花积分？添加你自己的 OpenAI、Google、Fal、Replicate、HuggingFace 等 API 密钥。密钥以 AES-256-GCM 加密存储。

**提示词增强** — 不知道怎么写好的提示词？在 Studio 点击「Append」，让 LLM 帮你改写。可选 5 种增强风格。

**图像逆向** — 看到一张喜欢的图？上传到「Image Reverse」，提取生成参数 — 风格标签、构图细节、可直接使用的提示词。

**角色卡片** — 在做系列作品？将角色保存为卡片（脸部、服装、全身视图），在多次生成中保持视觉一致性。

## 积分

新用户注册时获得免费积分。每个模型根据等级每次生成消耗 1–5 积分。每日免费积分会自动刷新。

| 等级 | 图像消耗 | 视频消耗 |
| ---- | -------- | -------- |
| 经济 | 1 积分   | 2 积分   |
| 标准 | 1–2 积分 | 3 积分   |
| 高级 | 2–3 积分 | 4–5 积分 |
