为什么移动端使用堆栈视图

自由画布在手机上难以精确拖拽和阅读。默认展示执行顺序与节点摘要，用户需要空间定位时再进入画布模式。

右侧面板在移动端的替代

Assistant 使用底部 sheet，Inspector 使用全屏 sheet；这样不会遮挡节点主操作，也能避开底部系统安全区。

实现切入点

先实现桌面右侧工作舱和节点状态，再做移动端堆栈视图。移动端不需要第一阶段完整支持自由画布编辑。
