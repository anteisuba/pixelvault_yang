Uses 4 credits · Same model, different seeds

5. 设计决策总结

入口方式 推荐 Option A（Split Button），不增加 UI 复杂度
结果区域 替换 Canvas 的 GenerationPreview → VariantGrid 2×2
选择交互 点击整个格子 = 选择；hover 显示 "Select as best" 按钮
选中态 ring-2 ring-primary + 左上角赤陶 badge "✓ Selected"
积分提示 下拉菜单每项显示消耗积分数
退出变体模式 下次点 Generate（非变体）自动回到 single 模式
