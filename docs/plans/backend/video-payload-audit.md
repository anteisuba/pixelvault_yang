## Current Body Constructors

| Path                                                        | Current behavior                                                                                                                                                                                               |
| ----------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/services/providers/fal.adapter.ts` inline queue        | Generic body for every video model: `prompt`, `aspect_ratio`, `duration` as string, optional `image_url`, optional defaults `negative_prompt`, `cfg_scale`, `prompt_optimizer`, `generate_audio`, `resolution` |
| `workers/execution/src/index.ts` FAL worker                 | Same generic FAL body as inline path                                                                                                                                                                           |
| `src/services/providers/replicate.adapter.ts` inline sync   | Generic prediction body: `{ model, input: { prompt, aspect_ratio, duration, image? } }`; not used by current video queue service because no Replicate video model exists                                       |
| `src/services/providers/volcengine.adapter.ts` inline queue | `POST /contents/generations/tasks` with `model`, `content`, `ratio`, `duration`, `resolution`, optional `generate_audio`, `return_last_frame`, `watermark`                                                     |

## Phase 2 Result

| Group                      | Models / modes                                                                                   | Result                                                                                                                                                          |
