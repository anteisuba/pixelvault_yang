## 一句话主基调

**「一个人的工作手记。」** PixelVault 是创作者摊开在桌上的手记：**人用手写下达意图，机器用正体排版执行，作品自己发色。** 整个产品就是「1 Person + AI = 1 Team」的纸面现场。

## 气质关键词

潦草但清醒 · 个人的、未修饰的 · 数字原生手绘（不是怀旧拟物）· 专业密度不打折 · 蓝墨水的温度

## 系统骨架：两个声音

这是整套设计的第一原理，所有细则由它推导：

| 声音       | 载体                                   | 视觉语言                                     |
| ---------- | -------------------------------------- | -------------------------------------------- |
| **人声**   | 标题、正文、批注、空态引导、图注       | 手写体 + 蓝墨水 + 手绘线（下划线/圈注/箭头） |
| **机器声** | 按钮、表单、参数、数据、进度、系统状态 | 正体无衬线 + 炭墨 + 精确几何                 |
| **作品**   | 生成的图/视频/音频                     | 主角；chrome 退后，不给作品加装饰            |

人声和机器声**永不混用**：控件上没有手写字，手写段落里没有系统灰话术。这条界限本身就是风格。

## 三支笔 — 语义色系统

手记里的人只有三支笔，UI 语义色就是这三支：

| 笔       | 语义                                        | 草案值                |
| -------- | ------------------------------------------- | --------------------- |
| **蓝墨** | 交互、选中、链接、焦点、人声强调            | 同 `--primary`        |
| **红笔** | 错误、破坏性操作、警示（手记里的划错/订正） | `oklch(52% 0.185 29)` |
| **墨绿** | 成功确认（克制使用，多数成功用蓝墨对勾）    | `oklch(50% 0.09 155)` |

数据本身永远炭墨；三支笔只做状态与语义，不做装饰。

## 图形语言 — 数字手绘五件套

取代「工坊宅邸」的装帧语言（裱框/贴入/藏书票作废；**盖章保留**，它在手记世界同样成立）：

1. **手绘下划线** — 人声强调（如当前步骤、关键词）
2. **圈注** — 聚焦引导（onboarding、新功能指认）
3. **手绘箭头** — 空态起手势指引（指向输入框/首个动作）
4. **对勾 / 叉** — 完成与失败的瞬时反馈（笔画自绘动画）
5. **索引标签** — 功能域辨识（见下节）

**红线（P0）**：手绘元素一屏 ≤2 处；单宽度墨线、平面化（Excalidraw 气质），**禁纸纹理贴图/胶带/回形针/咖啡渍等拟物**；控件与数据区禁手绘；生成作品上禁一切叠加装饰。

## 域辨识 — 索引标签制（颜料纪律的翻译）

「十房间六颜料」翻译成手记语言：**手记侧面的六色索引标签**。纪律三条原样继承：

1. 标签色只给六个创作域；档案四域（画廊/提示词/素材/角色卡）不执色，让藏品发色。
2. 标签色只出现在：导航标记 / 进度 / 生成态 / 空态插画 / 选中描边。
3. 永不做大面积彩底；一屏一支标签色。

| 域     | 标签色草案     | 备注                           |
| ------ | -------------- | ------------------------------ |
| 画布 ★ | 石绿 `#3D8A6B` | **沿用已落地值，不返工**       |
| 图片   | 青碧 `#2E8286` | 原群青撞蓝墨水，改青碧         |
| 视频   | 朱砂 `#C3402F` | 与红笔同相近色，落地时拉开明度 |

## 动效性格

笔画自绘（SVG stroke 240ms ease-out：对勾/下划线/圈注画出来）· 盖章顿挫（已有）· 纸片就位（沿用 motion 常量体系）。仍只服务状态澄清/连续性/反馈，禁装饰循环动效。

## 交互底线（继承 v1，不随方向变）

焦点环 · 键盘可达 · 自适应命中区（fine 32/36、coarse 44、绝对底线 24）· ResponsiveOverlay 披露 · 空态起手势 · 不支持不渲染 · 触屏软键盘只在直接点输入框时弹。
