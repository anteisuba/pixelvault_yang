## 一、 提示词核心要素构架 (Core Elements)

在撰写提示词时，必须包含以下核心维度的详细文字描述：
**镜头基础**：明确景别（如特写/中景/全景）、角度（如俯拍/仰拍）、运动方式（如推/拉/摇/移），并必须使用专业的影视拍摄技巧术语（例如希区柯克变焦）。
**镜头目的**：叙述该镜头的核心意图（如为了强调情绪、展示空间、或制造紧张感）。
**物理效果**：提供自然语言描述与物理效果的对应解释。
**光线质感**：明确是硬光还是柔光，设定光比，并描述高光部分是否溢出一层“氛围”。
**时间与节奏**：明确镜头的持续时间、剪辑方式（快切或长镜头）以及画面内的运动节奏。
**微表情与动作**：细致描述演员的微小表演（如眼神偏移、呼吸顿挫）；若为复数人物镜头，需明确每个人的动作是否分开写、是否有关联，以及是独立事件还是连续事件。

## 二、 镜头术语字典 (Terminology Dictionary)

### 1. 景别 (Shot Size)

**定义**：景别决定了观众能看到什么及看到多“近”，它是衡量画面与观众心理距离的尺子。

| 类别              | 英文术语               | 简短说明                                                                   |
| :---------------- | :--------------------- | :------------------------------------------------------------------------- |
| **大远景/大全景** | Extreme Long/Wide Shot | 环境为主，主体在画面中很小，用于交代宏大的场景和地理位置。                 |
| **远景**          | Long/Wide Shot         | 能完整展现人物全身，环境仍占较大比重，用于确立人物与环境的关系。           |
| **全景**          | Full Shot              | 人物从头到脚充满画面，主要强调人物的动作和肢体语言。                       |
| **中远景**        | Medium Long Shot       | 也称“3/4镜头”，从人物膝盖以上开始拍摄，介于中景和全景之间。                |
| **牛仔镜头**      | Cowboy Shot            | 中景的特殊变体，从人物大腿中部开始拍，源于美国西部片。                     |
| **中景**          | Medium Shot            | 最常用景别之一，通常从人物腰部以上拍摄，既表现人物也兼顾环境。             |
| **中特写**        | Medium Close-Up        | 介于中景和特写之间，从人物胸部或肩部以上拍摄，表情和环境并重。             |
| **特写**          | Close-Up               | 用主体的局部（如人脸）充满画面，用以强调细节和角色的强烈情绪。             |
| **项圈镜头**      | Choker                 | 特写镜头的变体，专门拍摄人脸，范围在眉骨以下、嘴巴以上，极致聚焦面部表情。 |
| **大特写**        | Extreme Close-Up       | 极度强调主体某个细微局部（如只拍眼睛或嘴部细节），带来极强冲击力。         |
| **意式镜头**      | Italian Shot           | 特指只拍摄眼部的大特写镜头，由意大利导演赛尔乔·莱昂内发扬光大。            |

**创作应用场景**：
**远景**：想展现主角站在山顶，背后是壮阔云海时使用。
**中景**：拍摄两个人对话，既要看到他们的动作，也要看清部分表情时使用。
**特写**：角色发现秘密时，瞳孔的瞬间放大，使用此镜头效果极佳。

### 2. 镜头角度 (Camera Angle)

**定义**：角度决定观众看故事的“高度”和“方向”，直接影响角色的力量感和画面情绪。

| 类别         | 英文术语               | 简短说明                                                         |
| :----------- | :--------------------- | :--------------------------------------------------------------- |
| **平拍**     | Eye Level Shot         | 最中性的角度，与人眼视线齐平，感觉自然客观，不带特殊情绪。       |
| **俯拍**     | High Angle Shot        | 从高处往下拍，让角色显得弱小、无助或受控制。                     |
| **仰拍**     | Low Angle Shot         | 从低处往上拍，使角色显得高大、强势、充满力量或威胁感。           |
| **斜角镜头** | Dutch Angle/Tilt       | 画面故意歪斜，制造不安、紧张、混乱或迷幻的视觉感受。             |
| **越肩镜头** | Over-the-Shoulder Shot | 越过一个角色的肩膀去拍另一个角色，常用于对话场景，非常有代入感。 |
| **主观视角** | Point-of-View (POV)    | 模拟角色眼中看到的世界，让观众“变成”那个角色，代入感极强。       |
| **鸟瞰镜头** | Bird's-eye View        | 从正上方俯瞰，仿佛上帝视角，极好地展示空间布局或让人物显渺小。   |

**创作应用场景**：
**俯拍**：表现被霸凌的孩子蜷缩在角落时使用。
**仰拍**：表现超级英雄从天而降，充满压迫感时使用。
**斜角镜头**：表现主角醉酒后踉跄、迷离的世界时使用。
**主观视角**：表现怪物透过猫眼窥视屋内的恐惧感时使用。

### 3. 镜头运动 (Camera Movement)

**定义**：引导视线、创造节奏并传递情绪，让静态画面获得生命力。

| 类别         | 英文术语                 | 简短说明                                                         |
| :----------- | :----------------------- | :--------------------------------------------------------------- |
| **推**       | Push In / Dolly In       | 摄影机物理靠近主体，景别由大变小，强调主体或细节。               |
| **拉**       | Pull Out / Dolly Out     | 摄影机物理远离主体，景别由小变大，展现主体与环境的关系。         |
| **摇**       | Pan (水平) / Tilt (垂直) | 摄影机位置不动，机身做水平或垂直转动，像人环顾四周。             |
| **移**       | Tracking / Trucking Shot | 摄影机在轨道或稳定器上水平移动拍摄，产生流动的空间感。           |
| **跟**       | Following Shot           | 摄影机跟随运动主体一起移动，始终让主体保持在画面中央。           |
| **甩**       | Whip Pan / Swish Pan     | 极快速的摇镜头，画面产生模糊效果，常用于快速转场。               |
| **升降**     | Boom / Crane Shot        | 摄影机在垂直方向上升或下降，改变视点高低，营造空间感或戏剧性。   |
| **手持**     | Handheld Shot            | 摄影机由摄影师手持，画面有呼吸感、真实感，常用于纪录片或惊悚片。 |
| **弧型运动** | Arc Shot                 | 摄影机围绕主体做弧形运动，保持距离不变，全方位展示主体或环境。   |
| **随机运动** | Random Movement          | 摄影机运动看似无规律，用来模拟混乱、焦虑的主观视角。             |

**创作应用场景**：
**推**：表现主角独自走向巨大未知的危险时使用。
**拉**：表现主角发现自己身处更庞大阴谋或场景时使用。
**跟**：表现主角在拥挤集市中穿行，营造混乱临场感时使用。
**甩**：从主角受惊的脸快速切换到导致其恐惧的事物时使用。

## 三、 进阶与特殊技法 (Advanced Techniques)

熟练掌握基础后可混合使用（如推轨+上升、跟拍+摇移）创造丰富的视觉语言。

**推轨变焦 (Dolly Zoom) / 希区柯克变焦 (Hitchcock Zoom)**
**定义**：摄影机向后或向前移动的同时，镜头焦距向反方向变焦，产生背景变形、主体不变的诡异效果。
**情感效果**：表现角色内心极度震惊、恐惧或眩晕，揭示真相。
**创作示例**：主角得知惊人秘密，感觉世界扭曲时的心理状态。
**主观视角镜头 (POV Shot)**
**定义**：模拟角色眼中看到的世界，观众直接代入角色视角。
**情感效果**：极强的临场感和代入感。
**创作示例**：从怪物或跟踪者视角看世界，营造悬疑偷窥感。
**固定镜头 (Static Shot)**
**定义**：摄影机完全静止无运动。
**情感效果**：沉静、客观或压抑，聚焦画面内表演与细节。
**创作示例**：表现主角等待结果时，放大时间的煎熬感。

## 四、 镜头提示词实战案例 (Prompt Examples)

**案例 1：多人动态场景与灯光切换**

> **【核心镜头指令】**
> 本段为第三人称客观电影级机位，画面完全融为一体（非左右分屏，非第一人称）。全局包含四次极其干净利落的硬切（Hard Cut）。
>
> **【全局设定】**
> 动漫三渲二风格，次世代三维游戏高精度渲染（Unreal Engine 5级别），电影级超高清画质，无字幕文字，无BGM。整体环境为唯美、梦幻的夜景暗调（Cinematic Night Scene）。极高品质的光线追踪（Ray Tracing），强调光源的衰减、漫反射以及人物面部的次表面散射（Subsurface Scattering）肌肤质感。人物设定：粉发少女（穿着带有黄丝带的校服，双马尾），黑发少年（穿着黑色夹克与半指手套）。

- **【0-4秒】中景**：夜景暗调与强烈的动态点光源。少年少女并肩挥舞冷烟花。冷烟花爆射出的金橘色火花为绝对主光源，照亮上半身。面部和锁骨处阴影高频动态游走（Dynamic Chiaroscuro），渲染欢快氛围。
- **【4-7秒】硬切镜头（Hard Cut）+ 特写镜头上摇（Tilt Up）**：镜头贴近女孩手部，伴随平滑阻尼感缓缓上摇至笑脸。极高对比度伦勃朗光，火花在脸颊跳跃，眼瞳反射火光，发丝呈现半透明透射质感（Translucency）。
- **【7-10秒】硬切镜头（Hard Cut）+ 低机位中景仰拍（Low Angle Shot）**：镜头降低仰拍，烟雾弥漫产生强烈的空气透视与体积光（Volumetric Lighting）。后方冷色路灯形成冷色逆光（Rim Light），与前景烟花暖金色漫反射（丁达尔效应）交汇。
- **【10-13秒】硬切镜头（Hard Cut）+ 男孩面部极近景特写（Extreme Close-up）**：极具戏剧张力的底光（Underlighting）由下至上照亮少年的下颌线与鼻梁。眼眸倒映火苗，背景陷入纯粹极暗阴影，聚焦微表情。
- **【13-15秒】硬切镜头（Hard Cut）+ 女孩面部极近景特写（Extreme Close-up）**：视线匹配（Eyeline Match）。光影瞬间变得柔和、均匀（无死角暖光补光）。消除生硬阴影，双马尾散发柔和光晕（Glow），定格极致明媚的幸福感。

**案例 2：极致微表情与呼吸感长镜头**

> **【全局设定】**
> 保持前序视频的动漫三渲二风格、次世代高精度渲染与极高品质光线追踪。所有镜头均带有轻微的手持摄影呼吸感（Subtle Breathing Camera Effect），画面焦点始终锁定在人物眼部。
>
> - **【0-5秒】黑发少年极近景特写**
>   - **镜头动态**：伴随呼吸频率，在Z轴上产生极缓慢细腻的推进感（Dolly in）。
>   - **0-2秒**：面部平静，下颌线紧绷，睫毛投下高频闪烁阴影，微风吹拂额前碎发。
>   - **2-4秒**：喉结微动，眼球平滑匀速转向左侧锁定画外。面部肌肉牵引：眼轮匝肌放松、嘴角极慢勾起。
>   - **4-5秒**：绽放温柔微笑，嘴唇边缘展现次表面散射透光质感。金橘色底光勾勒鼻梁与唇角。
> - **【5-15秒】硬切（Hard Cut）+ 粉发少女极近景特写**
>   - **镜头动态**：视线完美匹配（Eyeline Match），略带俯视感俯拍仰脸。长达10秒的镜头呼吸感配合情绪起伏呈现波浪感。
>   - **5-7秒**：察觉视线，猛然抬眼看向右侧。瞳孔微放大，嘴唇微张成“O”型，闪过错愕与惊喜。
>   - **7-10秒**：眼轮匝肌剧烈收缩，眼弯成月牙状，苹果肌上提。光影魔术转变：跳跃冷光融化为柔和、均匀、包裹式大面积暖光。
>   - **10-13秒**：头部向右偏倾（Head Tilt）约15度。风吹过粉色碎发。胸腔急促喜悦呼吸带动肩颈与黄丝带微震。
>   - **13-15秒**：深吸气，慢速眨眼一次，眼内高光闪烁。双马尾散发柔和光晕（Glow），沉浸在极致明媚与幸福的光影中。
