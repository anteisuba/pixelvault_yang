# QA Docs

Last updated: 2026-07-08

This directory is for reusable QA checklists and verification scripts.

Do not store one-off test reports here unless they have become reusable
guidance. Evidence files such as screenshots, JSON browser traces, or visual
captures belong under `docs/screenshots/` or the relevant `docs/design/reviews/`
asset folder.

## Use For

- Browser flow checklists
- Mobile and responsive QA checklists
- Regression checklists
- Manual verification scripts
- Known reusable test coverage gaps

## Do Not Use For

- Temporary task notes
- Raw logs from a single run
- Historical progress reports
- Active implementation plans

## Checklist Template

```md
# <Flow or Area> QA Checklist

## Scope

## Preconditions

## Steps

## Expected Evidence

## Known Gaps

## Source of Truth

## Last Verified
```
